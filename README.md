# WebStackGo
本项目使用 [Webstack网址导航](https://github.com/WebStackPage/WebStackPage.github.io) 作为前端框架，基于Go和Gin框架实现后端功能，努力做到轻量级与无依赖的网址导航网站。

未来将实现Docker轻量级部署。
